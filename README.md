# Simple AI Purchase Recommendation System

## Overview
A lightweight Python-based recommendation system that suggests products to customers based on their purchase history using AI/ML techniques.

## Features
- Customer purchase pattern analysis
- Product similarity recommendations
- Collaborative filtering
- Frequent pattern mining
- Customer behavior insights

## Technologies
- Python 3.6+
- Pandas & NumPy
- Scikit-learn

## Installation
```bash
pip install -r requirements.txt
python main.py
